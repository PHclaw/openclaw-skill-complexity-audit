# openclaw-skill-complexity-audit

代码复杂度评分与圈复杂度分析。扫描代码库，识别高复杂度函数（CC>10），给出重构建议。

## 功能

- 圈复杂度（CC）计算与评级
- 多语言支持（Python radon / JS ESLint / Go gocyclo）
- TOP N 高复杂度函数排序
- 重构模式推荐（卫语句、策略模式、提取函数）
- 代码行数（LOC）辅助分析

## 安装

将 `SKILL.md` 复制到 OpenClaw skills 目录即可。

## 复杂度评级

| CC | 评级 |
|----|------|
| 1-10 | ✅ 简单，可接受 |
| 11-20 | ⚠️ 中等，建议拆分 |
| 21-50 | 🔶 复杂，必须拆分 |
| > 50 | 🔴 极复杂，紧急重构 |
